FAQ

Comment changer les droits sur Linux ?

La commande chmod permet de modifier les droits et permissions.

Que veut dire la propriété NaN en Js ?

La propriété NaN signifie que la variable en question n’est pas un nombre. (Not a Number)

Est-il possible de modifier un commit ?

La commande git commit --amend permet de modifier le commit le plus récent.

Comment mettre à jour Ubuntu ?

sudo apt update / upgrade
